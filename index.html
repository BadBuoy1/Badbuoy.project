Other buoys float. Bad Buoys communicate.

What is the Bad Buoys project?

Bad Buoys is an open-source project creating low-cost, DIY, Raspberry Pi–powered maritime buoys. Designed for environmental and maritime monitoring, swarm communication, allowing modular nodes to bring modern tools to maritime security and research. We can create a maritime drone together that can be multi-purpose, inexpensive, and achievable for the novice technician or student. Let's protect our waterways together!

Core Ideas

  • Maritime monitoring and security – Buoys equipped with LoRa, camera and GPS can cross-check vessel AIS signals with real position data, flagging spoofed or ghost ships in restricted waters - added raspberry sensors allow for water treatment testing ,audio/video, live sonar data, Modular payloads (temperature, salinity, pH, pollution detectors) can be added to collect environmental and climate data in coastal zones.
  • Drift Tracking – Each buoy logs GPS data, allowing researchers or security teams to model ocean currents, vessel drift, or floating debris in real time.
  • Raspberry pi additional Sensors and Modular payloads – The raspberry pi board allows everything from sonar, wifi monitoring mode, audio/video - in reality the imagination is the limit on how a raspberry pi can be modified for different tasks, and the Bad Buoys project allows these ideas to become waterborn, communicative, and meshed. These applications range from environmental monitoring even in remote locations to reconnaissance and policing.
  • Swarm Mesh Networking – Buoys communicate with each other via LoRa, forming a self-healing mesh network that relays data back to shore even if one node fails.
