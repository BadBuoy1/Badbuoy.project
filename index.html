What is Bad Buoys?

Bad Buoys is an open-source project creating low-cost, DIY, Raspberry Pi–powered maritime buoys. Designed for environmental monitoring, AIS spoof detection, and swarm communication, these modular nodes bring modern tools to maritime security and research.

Core Capabilities

📡 AIS Spoof Detection
🌊 Drift Tracking
🧪 Environmental Sensors
🔗 Swarm Mesh Networking
🔋 Solar Recharge
