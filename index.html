What is the Bad Buoys project?

Bad Buoys are an open-source project creating low-cost, DIY, Raspberry Pi–powered maritime buoys. Designed for environmental and maritime monitoring, swarm communication, allowing modular nodes to bring modern tools to maritime security and research. We can create a maritime drone together that can be multi-purpose, inexpensive, and achievable for the novice technician or student.

Core Ideas

AIS Spoof Detection
Drift Tracking
Environmental Sensors
Swarm Mesh Networking
